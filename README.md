## 📋 About the Project

EgDev is a premium, conversion-optimized marketing website for a DevOps & Cloud engineering startup based in Egypt. The site is designed with a futuristic SaaS aesthetic inspired by Stripe, Vercel, and Linear.

**Key Features:**
- 🌙 Dark / Light mode toggle
- 🌍 Bilingual support — English & Egyptian Arabic (عربي مصري) with RTL layout
- Smooth scroll-reveal animations
- Hero mouse parallax interaction
- Infinite tech stack marquee
- Fully responsive (mobile, tablet, desktop)
- Lead capture form (Free Audit)

## 🌍 Bilingual System

The site supports full English ↔ Egyptian Arabic switching:

- Toggle button in the navbar (`EN` / `ع`)
- Language preference saved to `localStorage`
- Arabic mode activates RTL layout automatically
- Arabic font (Cairo) loads on switch
- All text elements use `data-en` / `data-ar` attributes for instant swap
